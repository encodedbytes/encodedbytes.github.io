### Prerequisites

- Ruby 2.7+
- Bundler
- Git

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/encodedbytes/encodedbytes.github.io.git
cd encodedbytes.github.io
```

2. Install dependencies:
```bash
bundle install
```

3. Run the development server:
```bash
bundle exec jekyll serve
```

4. Visit `http://localhost:4000` in your browser

## ğŸ¨ Customization

### Colors

Edit CSS variables in `assets/css/main.css`:

```css
:root {
    --primary-color: #3b82f6;
    --secondary-color: #10b981;
    --accent-color: #8b5cf6;
    /* ... */
}
```

### Content

- **Company Info**: Edit `_config.yml`
- **Page Content**: Edit `index.html`
- **Styles**: Modify `assets/css/main.css`
- **Scripts**: Update `assets/js/main.js`

### Contact Form

To enable the contact form, sign up at [Formspree](https://formspree.io/) and replace `YOUR_FORM_ID` in `index.html`:

```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
```
